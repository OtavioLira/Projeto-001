Documentação: Lira API CID-10

Introdução

Bem-vindo à Lira API CID-10, um dos projetos da meta de desenvolver 100 aplicações focadas em programação. Este projeto exemplifica a integração entre o framework Flask e o banco de dados NoSQL MongoDB, oferecendo uma API robusta para manipulação de dados da tabela CID-10 de forma totalmente gratuita. Este projeto ainda está em desenvolvimento, portanto, podem ocorrer incoerências ou atrasos na funcionalidade da plataforma. Agradecemos sua compreensão e feedback para melhorias contínuas.

Objetivo

A Lira API CID-10 foi criada para:

  • Demonstrar práticas modernas de desenvolvimento web utilizando Flask e MongoDB.
  • Facilitar o acesso aos dados de Classificação Internacional de Doenças (CID-10).
  • Servir como um exemplo didático na jornada de construção de 100 projetos completos.

Como Usar

Para utilizar a API, acesse algumas das as rotas disponíveis:

      • GET /cids (Pegar todos os cids):
        https://projeto-001.onrender.com/cids: Acessando essa rota você consegue visualizar todos os CIDs registrados em formato JSON, cada CID possui 3 informações; código, nome da doença e a categoria.
      • Para capturar um CID especifico existe 2 métodos possíveis, pelo nome da doença ou por código, siga abaixo para entender mais
        1. GET /cids/codigo/[código]:
          https://projeto-001.onrender.com/cids/codigo/ Retorna CIDs filtrados pelo código especificado, exemplo:
          1.Buscando o codigo CID E11,https://projeto-001.onrender.com/cids/codigo/E11 2.Buscando o codigo CID Z76.5,https://projeto-001.onrender.com/cids/codigo/Z76.5
        2. GET /cids/nome/[nome da doença]: Retorna CIDs filtrados pelo nome especificado.
          nota: o método de pesquisa ainda está simples, por isso caso o nome não esteja exatamente igual não terá o resultado esperado, recomendado que use a pesquisa por código
          1.Buscando o CID de Cólera, https://projeto-001.onrender.com/cids/nome/cólera 2.Buscando o CID de Amebíase, https://projeto-001.onrender.com/cids/nome/amebíase
        GET /cids/[inicio:fim] Também é possível retornar CIDs em um intervalo específico.
        Buscando os códigods CID de B00 até C00, https://projeto-001.onrender.com/cids/B00:C00
